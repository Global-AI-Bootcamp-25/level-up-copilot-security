# GitHub Agentic Workflows

This repository uses [GitHub Agentic Workflows](https://github.github.com/gh-aw/) to automate maintenance and documentation tasks. The following workflows are configured:

## Daily Documentation Updater

**File:** `.github/workflows/daily-doc-updater.md`

Runs on a daily schedule (or manually via `workflow_dispatch`) to scan merged pull requests and commits from the last 24 hours, identify documentation gaps, and automatically create pull requests with documentation updates.

- **Engine:** GitHub Copilot
- **Permissions:** Read-only access to repository contents, issues, and pull requests
- **Output:** Draft pull request with documentation changes, labeled `documentation` and `automation`, auto-assigned to `copilot` for review
- **Auto-merge:** Enabled with a 1-day expiry

To trigger the workflow manually, go to **Actions â†’ Daily Documentation Updater â†’ Run workflow**.

## Agentic Maintenance

**File:** `.github/workflows/agentics-maintenance.yml`

Runs every 2 hours on a schedule to close expired discussions, issues, and pull requests that were created by agentic workflows and have exceeded their configured expiry time.

- **Permissions:** Write access to discussions, issues, and pull requests
- **Schedule:** `37 */2 * * *` (every 2 hours)
- **Trigger:** Also available via `workflow_dispatch`

This workflow is auto-generated by the `gh-aw` toolchain and should not be edited manually. To regenerate it, run:

```bash
gh aw compile
```
